# goblin
Goblin. Team productivity monitor.

## Develop guide

### Prepare Development Environment

1. Install `golang`
1. Install `nodejs`
1. Install `npm`
1. Install `make`

### Get source code

1. `git clone --recursive git@github.com:ggoblin/goblin.git $GOPATH/src/github.com/ggoblin/goblin`

### Build and Test

1. `cd $GOPATH/src/github.com/ggoblin/goblin`
1. `make build`
1. `make `
